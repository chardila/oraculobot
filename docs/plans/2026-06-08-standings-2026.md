# 2026 Standings Sync Implementation Plan

> **For Claude:** REQUIRED SUB-SKILL: Use superpowers:executing-plans to implement this plan task-by-task.

**Goal:** Sync live 2026 World Cup group standings from football-data.org into Supabase every 30 minutes so the question bot can answer "¿cómo va el grupo A?" and similar queries.

**Architecture:** New `wc_standings_2026` table in Supabase populated via upsert at the end of every `check-results.ts` run (GitHub Actions, every 30 min). Table exposed in the question system by adding it to `ALLOWED_TABLES` and `WC_SCHEMA_PROMPT` in `wc-sql.ts`.

**Tech Stack:** TypeScript, Supabase REST API, football-data.org v4 API, Vitest

**Step 1: Add `supaUpsert` helper**

```typescript
async function supaUpsert(path: string, rows: Record<string, unknown>[], onConflict: string): Promise<void> {
  const url = new URL(`${SUPABASE_URL}/rest/v1/${path}`);
  url.searchParams.set('on_conflict', onConflict);
  const res = await fetch(url.toString(), {
    method: 'POST',
    headers: {
      'apikey': SERVICE_KEY,
      'Authorization': `Bearer ${SERVICE_KEY}`,
      'Content-Type': 'application/json',
      'Prefer': 'resolution=merge-duplicates',
    },
    body: JSON.stringify(rows),
  });
  if (!res.ok) throw new Error(`Supabase upsert ${path}: ${res.status} ${await res.text()}`);
}
```

**Step 2: Add `syncStandings` function**

After the `supaUpsert` helper, add:

```typescript
async function syncStandings(): Promise<void> {
  const res = await fetch(
    'https://api.football-data.org/v4/competitions/WC/standings?season=2026',
    { headers: { 'X-Auth-Token': FOOTBALL_DATA_TOKEN } }
  );
  if (!res.ok) {
    console.error(`standings fetch failed: ${res.status}`);
    return;
  }
  const { standings } = await res.json() as { standings: Array<{ group: string; table: Array<{
    team: { name: string };
    playedGames: number;
    won: number;
    draw: number;
    lost: number;
    goalsScored: number;
    goalsConceded: number;
    goalDifference: number;
    points: number;
  }> }> };

  const now = new Date().toISOString();
  const rows = standings.flatMap((group, _) =>
    group.table.map((entry, i) => ({
      group_name: group.group.replace('GROUP_', ''),
      position: i + 1,
      team: entry.team.name,
      played: entry.playedGames,
      wins: entry.won,
      draws: entry.draw,
      losses: entry.lost,
      goals_for: entry.goalsScored,
      goals_against: entry.goalsConceded,
      goal_difference: entry.goalDifference,
      points: entry.points,
      updated_at: now,
    }))
  );

  await supaUpsert('wc_standings_2026', rows, 'group_name,team');
  console.log(`✅ Standings sincronizados: ${rows.length} filas`);
}
```

**Step 3: Call `syncStandings` at the end of `main()`**

In the `main()` function, after the `for (const match of pending)` loop and before the closing brace, add:
